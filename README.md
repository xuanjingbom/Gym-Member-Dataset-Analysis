# Gym Members Exercise Analysis

## Overview

This project analyzes gym members' exercise tracking data using R.

The objective is to explore exercise habits, identify patterns in workout behavior, and perform statistical analysis on fitness-related variables.

## Tools Used

- R
- R Markdown
- Data Visualization
- Statistical Analysis
